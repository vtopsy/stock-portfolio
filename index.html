CSV Import Format
11-column format (header row optional): Col 1: Action — 1=Buy, -1=Sell, 0=Fee
Col 2: Exchange (SGX, NYSE, NASDAQ, LSE, HKEX…)
Col 3: Symbol / Ticker
Col 4: Units / Shares
Col 5: Price per unit
Col 6: Date (YYYY-MM-DD)
Col 7: Currency (USD, SGD, HKD…)
Col 8: Total before fees
Col 9: Total after fees
Col 10: Label / Name
Col 11: Notes (optional) Exchange → Country:
SGX→SG · NYSE/NASDAQ→US · LSE→GB · HKEX→HK · TSE→JP · XETRA→EU · ASX→AU
