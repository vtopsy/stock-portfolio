📧 EmailJS Digest

CSV Import Format
Column order (no header required): Col 1: Action — 1=Buy, -1=Sell, 0=Fee
Col 2: Exchange (SGX, NYSE, NASDAQ, LSE…)
Col 3: Symbol / Ticker
Col 4: Units
Col 5: Price per unit
Col 6: Date (YYYY-MM-DD)
Col 7: Currency
Col 8: Total before fees
Col 9: Total after fees
Col 10: Label / Name
Col 11: Notes (optional) Exchange → Country:
SGX→SG · NYSE/NASDAQ→US · LSE→GB · HKEX→HK · TSE→JP · XETRA→EU
EmailJS Template Variables
{{portfolio_value}} — Total value
{{total_pnl}} — Unrealized P&L
{{pnl_pct}} — P&L %
{{annual_div}} — Annual dividends
{{top_holdings}} — Top 5 summary
{{upcoming_divs}} — Next payouts
{{currency}} — Display currency
{{date}} — Report date
